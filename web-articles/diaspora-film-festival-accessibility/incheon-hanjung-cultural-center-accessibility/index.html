🚆 장소 가는 길

💡

1. 지하철

  • 가장 가까운 역 : 1호선 / 수인분당선 인천역 1번 출구
  • 도보 거리 : 약 8분 (인천아트플랫폼 바로 옆)
  • 이동 경로 :
    • 1호선 이용 시 → 경사로로 바로 출구 이동 가능 (엘리베이터 불필요)
    • 수인분당선 이용 시 → 엘리베이터로 지상층 이동 후 1번 출구
    • 1번 출구에서 우측 큰 길 따라 약 8분 직진 → 한중문화관 도착
  • 접근성 정보 :
    • 인천역 앞 횡단보도 건넌 직후 보도 폭 좁거나 턱 있는 구간 일부 있음
    • 이후 구간은 전반적으로 평탄

2. 버스

  • 인천아트플랫폼 앞 '중구청' 정류장 하차 후 도보 이동 (한중문화관과 매우 가까움)
  • 반대편 방면은 하버파크호텔 앞 정류장 하차 → 횡단보도 1회
  • 운행 노선 : 인천e음12, 간선 15·26, 좌석 307
  • 일부 저상버스 운행 노선 포함

3. 자차 이용 시 주차

  • 한중문화관 지하 주차장 이용 가능
  • 장애인 전용 주차구역 : 지하 1층 C구역 3면 + D구역 1면, 총 4면
  • 엘리베이터 인근에 위치
    • ⚠️ 주의 : 방문 당시 엘리베이터 점검 중이었음 (방문 전 확인 필수, 문폭 90cm)
  • 지하 주차장 → 엘리베이터 → 1층 하차 → 유리 여닫이문(폭 90cm, 무거움) → 약 6m 이동 → 한중문화관 정문 진입
    • ⚠️ 지하 주차장이 건물로 바로 연결되지 않음 (외부 이동 구간 있음)
  • 요금 : 최초 30분 1,000원 / 이후 15분당 500원 / 1일 최대 10,000원 / 24시간 무인운영
    • ⚠️ 인근 차이나타운은 항상 혼잡, 여유 있게 방문 권장

4. 장애인 콜택시 승하차

  • 추천 하차 위치 : 한중문화관 입구 (인천 중구 제물량로 236)

🎭 공연장 입장 및 휠체어석 정보

💡

1. 건물 외부 출입구

  • 건물 진입 구간 일부 보도 폭 좁아지는 구간 있음 (주차장 차량 진입로 이용 필요, 짧은 구간 울퉁불퉁)
  • 건물 앞 보도 상태 양호
  • 입구 양쪽에 휠체어용 경사로 설치
  • 출입문 : 바깥문 자동문 / 안쪽문 수동문 (한쪽 항상 열려 있음)
  • 문 폭 : 약 160cm
  • ✅ 휠체어 사용자 진입에 어려움 없는 편

2. 1층 로비 → 4층 공연장 이동

  • 1층 로비 진입 후 단차 없이 엘리베이터 탑승 가능
  • 통로 폭 충분
  • 4층 엘리베이터 하차 → 로비 넓음
  • 4층 공연장 로비에서 매표소·안내데스크 운영 (휠체어 접근 가능 / 자원활동가·스태프 상주)
  • 매점 없음

3. 공연장 입장

  • 주출입문 (중앙) : 계단 있어 휠체어 진입 불가 ❌
  • 휠체어 전용 출입문 : 1열 앞쪽 문 이용 (문 폭 150cm)
  • 문 무게 무거운 편 → 입장 시 직원이 개폐
  • 입퇴장 모두 앞쪽 문 이용으로 동선 용이

4. 휠체어석 위치 및 시야

  • 4층 상영관 맨 앞줄 좌측 2석
  • 스크린이 무대 깊숙이 배치되어 있어 앞줄이지만 시야 괜찮은 편
  • 비지정좌석 운영 → 동반자는 휠체어석 옆 또는 뒷열 착석 가능

5. 공연장 내부 통로

통로폭비고
중앙 통로110cm넓은 편, 단 중간에 계단 있음
벽면 통로75cm좁은 편 + 경사 있음
휠체어 출입구 경로—계단 없이 이동 가능

🚾 화장실 이용 정보

💡

장애인 화장실

✅
1~4층 동일 위치에 장애인 화장실 있음. 표지판 명확하고 찾기 쉬움.
  • 위치 : 1·2·3·4층 동일 위치, 일반화장실 옆
    • 장애인 화장실 남녀 공용
  • 4층 공연장 휠체어 출입문에서 약 16m
  • 수동 미닫이 문 / 내부 공간 넓은 편
  • 안전바 ✅ / 세면대 ✅ (높이 80cm)
  • 표지판 있어 위치 파악 용이
